// PRIVACY POLICY

This extension is built for a fast, minimal new tab experience. No personal information is collected, stored, or shared.

// Weather API

Live weather info is fetched via Open-Meteo API using your location coordinates. This data is used only in-session and never stored or logged.

// Theme & UI

The UI is styled in an aesthetic custom theme palettes inspired by various visual styles like Material UI.

// Clock & Display

The clock widget mimics the expressive layout seen in Pixel Widgets.

// Analytics & Storage

This extension uses no cookies, tracking scripts, or external analytics. Everything runs locally on your device.
